# CAIE 9618 Computer Science P4 Support Materials

This repository contains support materials for CAIE (Cambridge Assessment International Education) A-Level Computer Science paper 9618 P4.

## Contents

- `Characters.txt`: Character set information for the programming project

## Usage

These materials are intended to support students preparing for the CAIE 9618 Computer Science Paper 4 (Practical).

## Getting Started

1. Clone this repository
2. Review the materials
3. Use them as reference for your programming project
